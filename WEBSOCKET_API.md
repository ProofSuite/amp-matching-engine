**Websocket Endpoint**: `/socket`

message is send in order to provide

CANCEL_ORDER (client -> engine)

Payload:
```
{
	"channel": "order_channel",
	"message": 
	{
		"msgType": "CANCEL_ORDER",

      "hash":"",
			"userAddress": "0xefD7eB287CeeFCE8256Dd46e25F398acEA7C4b63"
      "signature":""
		}
	}
}
```
